Welcome to CTATrainlines Project Page.

This is an Apple App to display Chicago 'L' (subway) system. Users are able to select a different route and pick the station which provides the live train schedule information for that station. User is able to pick any spot in the map and find the closest route between the selected 'L' station to his/her current spot.It also connects to the iPhone MAP , with which you can do the complex tasks on routing, planning as so on.
