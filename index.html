Welcome to CTATrainlines Project Page.

This is a Apple App to display the Chicago 'L' (subway) systems. Users would be able to select a different route and stations which provides the train schedule information for each station. User can pick any spot in the map and find the closest route between the selected 'l' station to his/her current spot.It also connect to the iPhone MAP , with which you can do the complex task on routing , timing as so on.
